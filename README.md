# Contractor Executive Dashboard
A dashboard that unites project management overview and financial data into one dashboard for contractors. A large problem with modern platforms is their scope on project management or financial, so this all-in-one dashboard integrates data from supported platforms (Knowify, QuickBooks) to provide real-time insight into the important statistics that you NEED to know such as projects, cash and assets, and business health while using AI to help analyze.
